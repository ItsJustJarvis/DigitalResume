GGDB - Good Game Database

The goal of this project was to create an informative video game discovery application where you can find game details, reliable game reviews, and stay up to date on trends.

GGDB implements node/express to dynamically generate and/or inject video game data from multiple API sources. It has been created with responsiveness in mind and is equally accessible on computers, tablets and mobile devices.

Case Study available at: https://learndigital.dev/students/reeve-jarvis/

TradeFolio

Over the course of two months, our team worked to complete an application prototype for the Apprenticeship/Trades program at North Island College. This was a government funded project, hoping to improve the documentation and skill tracking experience for upcoming Trade Students.
